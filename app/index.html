GTR Check Delivery

Automação Industrial

Cliente

GTR Consultoria

Ano de conclusão

2019

Tecnologias utilizadas

Descrição

Sistema que automatiza o processo de separação e embarque de itens vendidos em uma indústria.

Integrado à um handheld, para leitura de código de barras das etiquetas de identificação dos itens, que contém informações como quantidade, peso e lote.

Possuindo também integração à um software ERP, recebendo uma lista com os itens a serem separados ou embarcados, e retornar os dados da checagem.

  • Automação

  • Leitura de código de barras
